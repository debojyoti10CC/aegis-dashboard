# Weilchain Counter Applet

A basic counter smart contract (applet) for Weilchain that demonstrates:
- State management
- Method calls with parameters
- Basic applet structure

## Features

- Increment counter
- Decrement counter
- Get current count
- Reset counter
- Set counter to specific value

## Supported Languages

According to Weilliptic docs, Rust, Golang, C++, and TypeScript are supported.
This implementation uses Rust for the applet.
